# Paper Guide — 学术论文导读 HTML 生成器

一个 [Claude Code](https://claude.ai/code) 技能（Skill），用于将学术论文 PDF 自动生成结构清晰、排版精美的单页 HTML 导读文档。

## 功能特性

- **PDF 论文解析**：自动读取 PDF 格式的学术论文，提取标题、作者、摘要、各章节内容
- **中文导读生成**：用中文撰写讲解，引用论文原文时保留原语言
- **精美 HTML 输出**：生成自包含的单页 HTML 文件，内联样式，无需外部依赖
- **MathJax 公式渲染**：支持 LaTeX 公式的在线渲染
- **章节导航**：自动生成可点击的目录和"回到顶部"浮动按钮
- **多种内容块**：原文引用块、导读讲解块、图表说明块，视觉区分清晰

## 安装

将本仓库克隆到 Claude Code 的技能目录中：

```bash
# 全局安装（所有项目可用）
git clone git@github.com:daixinwang/paper-guide-skill.git ~/.claude/skills/paper-guide

# 或项目级安装
git clone git@github.com:daixinwang/paper-guide-skill.git .claude/skills/paper-guide
```

## 使用方法

在 Claude Code 中，提供一篇 PDF 论文并使用以下任意触发方式：

```
# 直接触发
/paper-guide /path/to/paper.pdf

# 自然语言触发
帮我做个论文导读 /path/to/paper.pdf
解读一下这篇 paper
生成导读
帮我读论文
```

生成的 HTML 文件将保存在与 PDF 同目录下，文件名为 `[论文标题简写]-导读.html`。

## 导读结构

生成的导读文档通常包含以下部分：

1. **论文概览** — 标题、作者、一句话总结、关键贡献
2. **研究背景与动机** — 问题定义与研究动机
3. **相关工作** — 领域内相关研究简述
4. **方法/模型** — 核心技术方案的详细讲解
5. **实验与结果** — 实验设置、数据集、主要结果
6. **讨论与局限性** — 方法的优势与不足
7. **结论** — 论文总结
8. **总结评价** — 综合点评与影响分析

实际结构会根据论文本身的章节安排进行调整。

## 设计风格

- 浅色学术风格，白色内容卡片 + 浅灰背景
- 中文正文使用思源宋体，标题使用思源黑体
- 蓝色引用块（原文）、橙色讲解块（评注）、紫色图表说明块
- 响应式布局，移动端友好
